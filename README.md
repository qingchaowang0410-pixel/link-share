# 链接分享平台

一个本地运行的个人链接收藏 MVP。所有数据保存在本机 SQLite 文件里，单人使用。

## 功能

- **看板**：总链接数、未读数、本周新增、分类占比、Top 5 分享人、最近 10 条
- **添加链接**：粘贴 URL 自动按域名规则猜分类；手动填标题、简介、分享人、标签、备注
- **链接库**：按标题/URL/分享人/标签关键词搜索；按分类、已读状态筛选；标记已读 / 编辑 / 删除
- **分享人**：点任一名字查看 TA 分享过的全部链接

## 启动

**推荐：双击 [启动.command](启动.command)** — 自动后台启动并打开浏览器。停止时双击 [停止.command](停止.command)。

或手动：
```bash
python3 -m pip install --user -r requirements.txt   # 首次
python3 app.py                                       # 每次启动
# 浏览器打开 http://127.0.0.1:8000
```

## 团队共享（局域网）

服务监听在 `0.0.0.0`，同一 WiFi 的同事可以直接访问：

- 你的访问地址：`http://127.0.0.1:8000`
- 同事访问地址：`http://<你的局域网IP>:8000`（启动后弹窗会显示）

把 [给同事的说明.md](给同事的说明.md) 发给同事即可。

**前置条件**：
- macOS 防火墙允许 Python 入站（系统设置 → 网络 → 防火墙）
- 你的电脑不睡眠（设置 → 锁屏 → "显示器关闭时阻止自动进入睡眠"）
- 公司 WiFi 没开"客户端隔离"

每个用户首次访问会被要求填名字（存浏览器 localStorage），后续添加/修改链接会自动记录"由谁添加"。

## 数据文件

SQLite 数据库自动建在 `data/links.db`。**备份就是把这个文件拷走**。

## 自定义域名规则

打开 [domain_rules.py](domain_rules.py)，在 `DOMAIN_RULES` 字典里加一行即可：

```python
"example.com": "你的分类名",
```

修改 `CATEGORIES` 可调整可选分类。

## 技术栈

- 后端：FastAPI + SQLite
- 前端：Alpine.js + Tailwind（均走 CDN，无构建步骤）
- 依赖：仅 `fastapi` 和 `uvicorn`
